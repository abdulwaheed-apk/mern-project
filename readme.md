> **MERN Stack REST API (Express & MongoDB)** 

**Install all the dependencies**
***
```
npm install
```
*Entry point server.js*

**To run server**
***
```
npm run server
```
open 
[localhost:5000](http://localhost:5000/)

 ***

``````````````
